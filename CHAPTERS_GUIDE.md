# チャプター作成ガイド (Chapter Creation Guide)

PacificTubeでは、YouTubeのようにビデオにチャプター（タイムスタンプ付きの目次）を追加できます。

## チャプターファイル形式

各ビデオに対応する `.chapters.json` ファイルを作成します。

### ファイル形式

```json
{
    "video": "土木技術基礎講座動画（構造力学）/構造力学（第1回）.mp4",
    "chapters": [
        {"timestamp": "0:00", "title": "イントロダクション"},
        {"timestamp": "5:30", "title": "曲げ剛性の影響（ボックスカルバート）"},
        {"timestamp": "15:00", "title": "演習−曲げ剛性の影響（アンダーピニング）"},
        {"timestamp": "30:00", "title": "弾性と塑性"},
        {"timestamp": "45:00", "title": "演習−弾性と塑性"}
    ]
}
```

### タイムスタンプ形式

- `"0:00"` - 分:秒
- `"5:30"` - 分:秒
- `"1:25:00"` - 時:分:秒

### フィールド説明

| フィールド | 必須 | 説明 |
|-----------|------|------|
| `video` | はい | ビデオファイルのパス（blob storage内） |
| `chapters` | はい | チャプターの配列 |
| `timestamp` | はい | チャプターの開始時間 |
| `title` | はい | チャプターのタイトル（スライドタイトルなど） |
| `description` | いいえ | 追加説明（オプション） |

## チャプターファイルの作成手順

### 1. ビデオを確認

ビデオを再生しながら、各セクションの開始時間とタイトルをメモします。

### 2. JSONファイルを作成

`chapters/` フォルダに `.chapters.json` ファイルを作成します：

```
chapters/
├── 構造力学（第1回）.chapters.json
├── 構造力学（第2回）.chapters.json
└── ...
```

### 3. アップロード

```powershell
python upload_chapters.py
```

これにより、`chapters/` フォルダ内のすべての `.chapters.json` ファイルがAzure Blob Storageにアップロードされます。

## 自動生成 vs 手動チャプター

| 方式 | 特徴 |
|------|------|
| **手動チャプター** | 正確なタイトル（スライドタイトルなど）を表示。推奨。 |
| **自動生成** | 字幕から5分間隔でテキストを抽出。手動ファイルがない場合のフォールバック。 |

手動チャプターファイル（`.chapters.json`）がある場合、そちらが優先されます。

## トラブルシューティング

### チャプターが表示されない

1. ファイル名が正しいか確認: `ビデオ名.chapters.json`
2. JSONが有効か確認: https://jsonlint.com/
3. `video` パスがblob storage内のパスと一致しているか確認

### チャプターのアップロードに失敗

```powershell
# 特定のファイルをアップロード
python upload_chapters.py chapters/構造力学（第1回）.chapters.json
```
